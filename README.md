# AI Website Copy Generator for Local Business
🚀Built as a scalable AI content generation workflow for real-world business applications.

## 📌 Overview
This project generates website content for local businesses using AI.  
It helps cafes and small businesses create professional website copy instantly.

## 🎯 Problem
Small businesses struggle to write professional website content.
Hiring copywriters is expensive.

## 💡 Solution
This project uses AI + structured prompts to generate high-quality website copy instantly.
An AI-powered system that generates:
- Homepage content
- Services section
- About section
- Call-to-actions

## 👥 Target Users
- Cafe owners
- Freelancers
- Small business owners
