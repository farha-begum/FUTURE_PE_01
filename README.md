# AI Website Copy Generator for Local Businesses

## 📌 Overview
This project generates website content for local businesses using AI.  
It helps cafes and small businesses create professional website copy instantly.

## 🎯 Problem
Many local businesses struggle to write effective website content.

## 💡 Solution
An AI-powered system that generates:
- Homepage content
- Services section
- About section
- Call-to-actions

## 👥 Target Users
- Cafe owners
- Freelancers
- Small business owners

## ☕ Business Example
Urban Cafreash – A cozy cafe designed for students and freelancers.

## ✨ Features
- AI-generated website content
- Reusable prompt system
- SEO-friendly copy
- Fast content generation

## 🛠 Tools Used
- ChatGPT
- Prompt Engineering
- GitHub

## ⚙️ How It Works
1. Input business details
2. Use prompts
3. Generate content instantly
